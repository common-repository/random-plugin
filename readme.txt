== Description ==

(Based on the now famous "Hello Dolly" plugin created by Matt Mullenweg.)
In the admin panel, in the upper right corner will show random plugins and a link to this plugin, where you can download it and read the detailed description of this plugin. The database has 360 plugins, they will be constantly replenished, within one month of planning to add all the plugins.

== Installation ==

To set the plugin up, you need to:

1. Upload `randomplugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

That's all!
